1. What is the symbol for Conjunction?



✅ The symbol for Conjunction is p ∧ q (and).

2. If p = True and q = False, what is p → q (Conditional)?



✅ The answer is False. A conditional is only false when p is True and q is False.

3. What connective is used for Biconditional and when is it True?



✅ The connective is "if and only if" (p ↔ q). It is True only when both p and q have the same value (both True or both False).
